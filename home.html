2001-2023 Chicago Crime Analysis

This project delves into Chicago's crime dynamics through the analysis of almost 8 million crime records spanning 2001-2023. Key findings include decreasing overall crime and arrest rates, a spike in domestic crimes during the pandemic, and the prevalence of property offenses.
It leverages Python for web scraping, SQL for exploration and analysis, and Power BI for dashboard creation.
